# Quick Task 260704-2w3: Уведомления при получении сообщения

## Задача

При отправке сообщения в мессенджере получатель должен получать уведомление (колокольчик, SSE, web push).

## План

1. Добавить `MESSAGE_RECEIVED` в `NotificationType` и миграцию Prisma.
2. Расширить domain events и `buildNotificationsForEvent` для сообщений.
3. При POST `/api/conversations/[id]/messages` — `dispatchDomainEvent` + `deliverNotifications`.
4. Ссылка в уведомлении → `/messages?conversation={id}`; открытие диалога по query param.
