# Quick Task 260625-0pl: Разлогин преподавателя при неактивности 1 час

## Цель

Автоматически разлогинивать только пользователей с ролью `TEACHER` после 1 часа бездействия. Остальные роли (MANAGER, SUPER_ADMIN, STUDENT) не затрагиваются.

## Задачи

### Task 1: IdleSessionGuard для учителей

**files:**
- `package.json` — добавить `react-idle-timer`
- `src/features/auth/lib/idle-session.ts` — константа таймаута
- `src/features/auth/ui/IdleSessionGuard.tsx` — клиентский guard
- `src/widgets/app-shell/ui/AppShell.tsx` — подключить guard

**action:** Установить `react-idle-timer`, создать guard с `useIdleTimer` (timeout 1ч, disabled для role !== TEACHER), вызов `signOut({ callbackUrl: '/login?reason=idle' })`.

**verify:** `pnpm lint` проходит; guard рендерится в AppShell.

**done:** Учитель получает idle logout, другие роли — нет.

### Task 2: Сообщение на странице входа + тесты

**files:**
- `src/app/(auth)/login/page.tsx` — передать `reason` в форму
- `src/features/auth/ui/LoginForm.tsx` — Alert при `reason=idle`
- `src/features/auth/lib/idle-session.test.ts` — unit-тест хелпера
- `e2e/auth.spec.ts` — E2E: idle logout учителя (clock mock)

**action:** Показать «Сессия завершена из-за неактивности» на login; покрыть логику и сценарий с Playwright clock.

**verify:** `pnpm test:unit` и целевой e2e spec проходят.

**done:** UX и тесты на месте.
