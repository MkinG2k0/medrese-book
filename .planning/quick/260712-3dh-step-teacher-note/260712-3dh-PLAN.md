# Quick Task 260712-3dh: Заметка учителя в шагах

## Задача

Добавить поле «Заметка учителя» в шаги программы — тот же Tiptap-редактор (StepContent), что и «Содержание». Поле видно учителям в журнале, не показывается ученикам.

## План

1. Prisma: `teacherNote Json` в модели Step + миграция
2. Zod-схема и server actions (create/update с валидацией)
3. UI админки: второй StepEditor в StepForm
4. Журнал: lazy-load и отображение через StepContentPreview в StepCard
