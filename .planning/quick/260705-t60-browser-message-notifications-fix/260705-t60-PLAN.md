# Quick Task 260705-t60: Браузерные уведомления для сообщений

## Проблема

Колокольчик обновляется (SSE), но системные уведомления не показываются при `Notification.permission === 'granted'`.

Причина: разрешение браузера ≠ push-подписка (`PushManager.subscribe` + запись в БД). UI скрывал CTA при `granted`, хотя подписки не было. SSE не вызывал `new Notification()`.

## План

1. Авто-подписка на push при `granted` без подписки (`usePushAutoSubscribe` в `NotificationBell`).
2. SSE: передавать title/body/link и показывать `Notification` при открытой вкладке.
3. Исправить `PushSubscribePrompt` и `PwaInstallBanner` — проверять реальную push-подписку.
