# Quick Task 260712-12i: Fix duplicate React keys in LevelStats Table

## Problem

`LevelStatsChart` uses `rowKey="level"` where `level` is `Level.number`. Номер уникален только внутри предмета (`@@unique([subjectId, number])`). Без фильтра по группе `getLevelStats` возвращает уровни всех предметов — дубликаты `1`, `2`, …

## Tasks

### 1. Добавить уникальный ключ и label в LevelStats
- `level-stats.ts`: включить `subject` в запрос, вернуть `levelId` и `label`
- `label`: «N» при одном предмете, «N (Предмет)» при нескольких

### 2. Обновить UI
- `LevelStats.tsx`: `rowKey="levelId"`, колонка и график по `label`
